## Key Table Descriptions

### Reservations & Charges
- **reservations** — Main booking record linking guest, room, dates, status, and pricing tier
- **reservation_charges** — Line-item charges (accommodation per season, tourism tax, parking, pets, short-stay supplement). Generated by `UnifiedPricingService`
- **reservation_statuses** — Status codes: `confirmed`, `checked-in`, `checked-out`, `room-closure`, `unallocated`, `incomplete-payment`
- **booking_sources** — Where the booking came from: `booking.com`, `direct`, `other`
- **labels** — Color-coded tags for reservations (VIP, corporate, etc.)

### Pricing
- **pricing_seasons** — Seasonal periods A (low/winter), B (shoulder), C (summer), D (peak Jul-Aug)
- **pricing_tiers** — Rate structures with seasonal rates A-D and optional percentage discounts
- **room_pricing** — Per-room, per-season base rates

### Financial
- **invoices** — Generated from reservation charges, supports R1 (company) billing
- **payments** — Payment records (cash, card, bank transfer, Booking.com)
- **fiscal_records** — Croatian tax authority (e-Racuni) JIR/ZKI codes and QR data
