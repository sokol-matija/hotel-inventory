## Prerequisites

Before you begin, ensure you have:

- **Node.js** 16+ (18+ recommended)
- **npm** or **yarn** package manager
- **Git** for version control
- **Supabase Account** (free tier available)

<Note>
For Croatian fiscalization features, you'll need a FINA P12 certificate (demo or production).
</Note>

## 5-Minute Setup

### Step 1: Clone and Install

```bash
# Clone the repository
git clone https://github.com/your-username/hotel-inventory.git
cd hotel-inventory

# Install dependencies
npm install
```

### Step 2: Environment Configuration

Copy the sample environment file:

```bash
cp .env.sample .env.local
```

Configure your `.env.local` file:

```bash
# Supabase Configuration (Required)
REACT_APP_SUPABASE_URL=https://your-project.supabase.co
REACT_APP_SUPABASE_ANON_KEY=your_anon_key_here

# Croatian Fiscalization (Optional - for invoice generation)
REACT_APP_FISCAL_CERT_PASSWORD=your_certificate_password
REACT_APP_FISCAL_OIB=87246357068  # Hotel OIB

# Channel Manager (Optional - for OTA integration)
REACT_APP_PHOBS_API_KEY=your_phobs_api_key
REACT_APP_PHOBS_API_SECRET=your_phobs_secret
REACT_APP_PHOBS_HOTEL_ID=your_hotel_id
```

<Tip>
Get your Supabase credentials from your project dashboard at [supabase.com](https://supabase.com)
</Tip>

### Step 3: Database Setup

The database schema is managed through Supabase migrations:

```bash
# Run migrations via Supabase Dashboard
# Navigate to: Database > Migrations in your Supabase project
# Or use Supabase CLI:
supabase db push
```

### Step 4: Launch Application

```bash
npm start
```

## First Steps After Launch

### 1. Authentication

<Steps>
  <Step title="Navigate to Login">
    Open `http://localhost:3000` in your browser
  </Step>

  <Step title="Sign In">
    Use Google OAuth or create an email/password account
  </Step>

  <Step title="Access Dashboard">
    You'll be redirected to the hotel management dashboard
  </Step>
</Steps>

### 2. Create Your First Reservation

<Steps>
  <Step title="Open Front Desk Module">
    Navigate to **Hotel** > **Front Desk** from the sidebar
  </Step>

  <Step title="Use Drag-to-Create">
    Click and drag on the timeline calendar to create a new booking
    - Drag horizontally to set date range
    - Drop on a room to assign
  </Step>

  <Step title="Enter Guest Details">
    Fill in the reservation modal:
    - Guest name and contact info
    - Number of adults/children
    - Special requests
  </Step>

  <Step title="Save Reservation">
    Click **Create Reservation** to save
  </Step>
</Steps>

<Tip>
**Keyboard Shortcut**: Press `N` anywhere in the front desk to create a new reservation
</Tip>

### 3. Generate Your First Invoice

<Steps>
  <Step title="Navigate to Finance">
    Go to **Hotel** > **Finance** from the sidebar
  </Step>

  <Step title="Select a Reservation">
    Find a completed reservation and click **Create Invoice**
  </Step>

  <Step title="Fiscalize (Croatian Only)">
    Click **Fiscalize Invoice** to generate JIR/ZKI codes
    - Requires FINA certificate configuration
    - Generates QR code for verification
  </Step>

  <Step title="Download PDF">
    Click **Download PDF** to generate invoice with fiscal data
  </Step>
</Steps>

## Key Integrations Setup

### Croatian Fiscalization

<Steps>
  <Step title="Obtain FINA Certificate">
    Get a P12 certificate from FINA (demo or production)
  </Step>

  <Step title="Configure Certificate">
    Place certificate in `p12 cert/` directory
    ```bash
    # Set password in .env.local
    REACT_APP_FISCAL_CERT_PASSWORD=your_password
    ```
  </Step>

  <Step title="Deploy Edge Function">
    Deploy the fiscalization Edge Function to Supabase
    ```bash
    supabase functions deploy fiscalize-invoice
    ```
  </Step>

  <Step title="Test Fiscalization">
    Use the Finance module to generate a fiscal invoice
  </Step>
</Steps>

### Channel Manager (Phobs)

<Steps>
  <Step title="Get Phobs API Credentials">
    Sign up at Phobs and obtain API keys
  </Step>

  <Step title="Configure Environment">
    Add credentials to `.env.local`
    ```bash
    REACT_APP_PHOBS_API_KEY=your_key
    REACT_APP_PHOBS_API_SECRET=your_secret
    REACT_APP_PHOBS_HOTEL_ID=your_hotel_id
    ```
  </Step>

  <Step title="Authenticate">
    Navigate to Channel Manager and click **Connect to Phobs**
  </Step>

  <Step title="Enable Channels">
    Select OTA platforms to sync (Booking.com, Expedia, etc.)
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Application won't start">
    **Solution**: Verify dependencies and environment variables
    ```bash
    # Reinstall dependencies
    rm -rf node_modules package-lock.json
    npm install

    # Check environment file
    cat .env.local

    # Start with verbose logging
    npm start
    ```
  </Accordion>

  <Accordion title="Supabase connection errors">
    **Solution**: Verify your Supabase credentials
    ```bash
    # Check Supabase project status
    # Visit: https://app.supabase.com/project/_/settings/api

    # Ensure REACT_APP_SUPABASE_URL and REACT_APP_SUPABASE_ANON_KEY are correct
    ```
  </Accordion>

  <Accordion title="Croatian fiscalization not working">
    **Solution**: Check certificate and Edge Function
    ```bash
    # Verify certificate password
    # Check Edge Function logs in Supabase Dashboard

    # Test locally
    node scripts/validate-zki-algorithm.js
    ```
  </Accordion>

  <Accordion title="TypeScript compilation errors">
    **Solution**: Build the project to see errors
    ```bash
    npm run build

    # TypeScript should compile with zero errors
    # Only ESLint warnings expected (unused variables)
    ```
  </Accordion>

  <Accordion title="Finance UI shows no invoices">
    **Solution**: Check database connection and data
    ```bash
    # Verify invoices exist in database
    # Use Supabase Dashboard > Table Editor > invoices

    # Check browser console for errors
    # Verify loadInvoices() is called in SupabaseHotelContext
    ```
  </Accordion>
</AccordionGroup>
