## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (version 18 or higher)
- **npm** or **yarn**
- **Git**
- **Supabase CLI** (optional, for database management)

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/hotel-inventory.git
cd hotel-inventory
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Environment Configuration

Copy the sample environment file and configure your settings:

```bash
cp .env.sample .env
```

Edit the `.env` file with your configuration:

```bash
# Supabase Configuration
REACT_APP_SUPABASE_URL=your_supabase_url
REACT_APP_SUPABASE_ANON_KEY=your_supabase_anon_key

# Optional: Croatian Fiscalization
REACT_APP_FINA_DEMO_MODE=true
REACT_APP_FINA_CERT_PATH=./certificates/demo.p12
REACT_APP_FINA_CERT_PASSWORD=demo_password
```

### 4. Database Setup

Run the database migrations to set up your schema:

```bash
npm run supabase:migrate
```

### 5. Start Development Server

```bash
npm run dev
```

Your application will be available at `http://localhost:3000`.

## Key Features Overview

### Real-time Timeline

The front desk timeline provides:
- **Drag & Drop**: Move reservations between dates and rooms
- **Conflict Detection**: Automatic validation prevents double bookings
- **Live Updates**: Real-time synchronization across all users

### Channel Manager Integration

Connect with major OTA platforms:
- Booking.com
- Expedia
- Airbnb
- And more...

### Croatian Fiscalization

Built-in support for Croatian fiscal requirements:
- FINA integration
- Automatic invoice generation
- Tax compliance reporting

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Application won't start">
    Check that all environment variables are set correctly and dependencies are installed.
    ```bash
    npm install
    npm run dev
    ```
  </Accordion>

  <Accordion title="Database connection issues">
    Verify your Supabase URL and keys in the `.env` file.
    ```bash
    # Check Supabase connection
    npm run supabase:status
    ```
  </Accordion>

  <Accordion title="TypeScript errors">
    Regenerate types from your database schema:
    ```bash
    npm run supabase:types
    ```
  </Accordion>
</AccordionGroup>
