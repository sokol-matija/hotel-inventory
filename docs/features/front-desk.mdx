## Timeline Interface

### Visual Layout

The timeline displays:
- **14-day view** - Configurable date range (7/14/30 days)
- **46 rooms** - All Hotel Porec rooms organized by floor
- **Color-coded status** - Visual indicators for reservation states
- **Real-time updates** - Live synchronization across all users

### Reservation States

```typescript
type ReservationStatus =
  | 'confirmed'          // Blue - Booking confirmed
  | 'checked-in'         // Green - Guest in room
  | 'checked-out'        // Gray - Completed stay
  | 'cancelled'          // Red - Cancelled booking
```

### Optimistic Updates

Instant UI feedback with automatic rollback:

```typescript
// Optimistic update flow
1. User creates reservation
2. UI updates immediately (optimistic)
3. Database save happens in background
4. If save fails → automatic UI rollback
5. User sees error notification

// Implementation
const optimisticUpdate = new OptimisticUpdateService();

await optimisticUpdate.createReservation(reservation, {
  onSuccess: () => showNotification('Reservation created'),
  onError: () => {
    // Automatic rollback to previous state
    showNotification('Failed to create - please try again');
  }
});
```

## Conflict Detection

### Real-time Validation

The system prevents double bookings:

<Steps>
  <Step title="Date Range Check">
    When creating/editing reservation:
    ```typescript
    const conflicts = await ConflictDetectionService.checkConflicts({
      roomId: selectedRoom.id,
      checkIn: selectedCheckIn,
      checkOut: selectedCheckOut
    });
    ```
  </Step>

  <Step title="Conflict Detection">
    Detects overlapping bookings:
    ```typescript
    // Conflict exists if:
    (newCheckIn < existingCheckOut) &&
    (newCheckOut > existingCheckIn)
    ```
  </Step>

  <Step title="Alternative Suggestions">
    If conflict found, suggest alternatives:
    ```typescript
    const alternatives = await findAlternativeRooms({
      roomType: currentRoom.type,
      checkIn, checkOut,
      capacity: numberOfGuests
    });

    // Returns rooms with:
    // - Same type/capacity
    // - Available for requested dates
    // - Similar or lower price
    ```
  </Step>

  <Step title="Visual Feedback">
    Red overlay on conflicting reservations
    - Warning modal with details
    - One-click alternative room selection
  </Step>
</Steps>
