## Keyboard Shortcuts Cheat Sheet

<Accordion title="üìã Complete Shortcuts Reference" defaultOpen={true}>

### Navigation
- `Cmd/Ctrl + K` - Command palette
- `F` - Focus search
- `‚Üê/‚Üí` - Previous/Next day
- `T` - Go to today
- `Esc` - Close dialog/Cancel

### Views
- `1` - Day view
- `2` - Week view
- `3` - Month view
- `E` - Expand timeline
- `C` - Compact timeline

### Actions
- `N` - New reservation
- `Cmd/Ctrl + S` - Save
- `Cmd/Ctrl + Z` - Undo
- `Cmd/Ctrl + Shift + Z` - Redo
- `/` - Search

### Selection
- `Cmd/Ctrl + A` - Select all
- `Shift + Click` - Multi-select
- `Cmd/Ctrl + Click` - Toggle selection

### Quick Status Changes
- `I` - Check in
- `O` - Check out
- `X` - Cancel
- `?` - Help

</Accordion>
